# image-matching-akaze

## 仮想環境（Windows）
参考：https://qiita.com/fiftystorm36/items/b2fd47cf32c7694adc2e  

### 作成
```
# python 3であることを確認
python -V
python -m venv [envname]
```
- `[envname]`: `py39`を指定

### アクティベート
```
.\py39\Scripts\activate
```
> (py39) PS C:\Users\image-matching-akaze>  

などとなることを確認する。

### pip install
```
pip install -r requirements.txt
```
